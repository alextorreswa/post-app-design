🚀 Progress Update – MindBridge (Mental Wellness App)

Over the past few weeks in my Application Development BAS program, I’ve been working on a mobile app called MindBridge — a mental wellness tool designed to help users manage stress, track emotions, and build daily mindfulness habits.

🧠 Project Overview

MindBridge focuses on creating a simple and calming experience where users can:

Log their daily mood
Practice breathing and grounding exercises
Visualize emotional trends over time

The goal is to make mental health support accessible and easy to use in under a minute.

👨‍💻 My Role

I worked as the UI/UX Designer and Frontend Developer, responsible for:

Creating the design system (colors, typography, components)
Designing high-fidelity mockups in Figma
Building a clickable prototype
Structuring the user experience across multiple screens
🛠️ Work Completed

During this phase, I:

Designed a complete visual style guide
(color palette, typography hierarchy, button styles)
Created high-fidelity screens including:
Home (mood tracking + streaks)
Mood input with emoji selection
2-minute breathing reset
Insights dashboard with charts
Tools section with guided exercises
Ensured consistency in spacing, alignment, and UI components
Applied accessibility principles, improving contrast and readability
⚠️ Challenges & Solutions

One challenge was ensuring color accessibility and contrast.
Some color combinations looked visually appealing but failed accessibility standards.

✅ Solution:
I used contrast-checking tools and adjusted colors to meet WCAG guidelines while maintaining the app’s calm aesthetic.

Another challenge was maintaining visual consistency across screens.

✅ Solution:
I created reusable components and followed a structured design system.

📚 Key Learnings
How to build a scalable design system
Importance of visual hierarchy and spacing
Applying color theory to create emotional impact
Designing for real users, not just visuals
🎯 Reflection

This project helped me understand how design decisions directly impact user experience and emotional engagement.

I’m especially proud of how the app communicates a sense of calm, trust, and simplicity.
